状态

git 有分区的概念，平时我们新建、删除、修改文件都是在工作区，通过 git add 命令把工作区的变动提交到暂存区（stage），再用 git commit 把stage的内容提交到分支（branch）。

每次我们对文件进行修改，都会改变 git 的状态。
