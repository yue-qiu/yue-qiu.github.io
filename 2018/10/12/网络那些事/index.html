网络那些事儿

发表于 2018-10-12 | Edited on 2018-10-13 | 分类于 计算机网络 | Comments： 0 | Views： | Views：

IPv4

遵循IPv4协议的报文首部多为20个 byte，其中目标IP地址与源IP地址各占 32bit。当路由器发现数据报大小超过链路 MTU 时会进行分片。

IPv4 地址采用点分十进制记法：xxx.xxx.xxx.xxx。IP地址对应的是网络接口而不是某个设备。

由于不同 IP 地址的高位可能是相同的，如：172.33.4.1 和 172.33.4.5。通常这表示使用这些地址的接口处于同一个网络中，而相同的高位被称为网络号(NetID)，不同的地位称为主机号(HostID)。具有相同网络号的接口组成了一个IP子网。

分类 IP 地址

IP 地址也是有分类的，根据网络号大小可以将 IP 地址分为5类。其中A, B, C 类最常见，占了整个 IPv4 地址的 85%。

  • A 类(0xxxxxxx, xxxxxxxx, xxxxxxxx, xxxxxxxx)：8位NetID，24位HostID。范围：0.0.0.0 —— 127.255.255.255
  • B 类(10xxxxxx, xxxxxxxx, xxxxxxxx, xxxxxxxx)：16位NetID，16位HostID。范围：128.0.0.0 —— 191.255.255.255
  • C 类(110xxxxx, xxxxxxxx, xxxxxxxx, xxxxxxxx)：24位NetID，8位HostID。范围：192.0.0.0 —— 223.255.255.255
  • D、E 类无NetID、HostID

私有 IP 地址

私有地址公网不可见，只用于内部网络。在公网上这些IP地址不会被使用

  • A类：NetID= 10，如：10.23.35.1
  • B类：NetID= 172.16 —— 172.31，如 172.16.34.1
  • C类：NetID= 192.168.0 —— 192.168.255 如 192.168.168.42

子网掩码

子网掩码确定网络号、子网号、主机号，长度为 32bit。路由器通过将子网地址与子网掩码进行与运算确定子网大小

子网掩码计算：IP地址网络号、子网号位全置 1，主机号置 0。A类掩码 255.0.0.0，B类 255.255.0.0，C类 255.255.255.0

例：B类划 3bit 为子网号，如 128.255.224.3(10000000, 11111111, 11100000, 00000011)，则掩码为 255.255.224.0（11111111, 11111111, 11100000, 00000000）

DNS劫持与污染

DNS 污染是指服务器检测到用户访问特定的主机时，伪装成 DNS 服务器向用户返回错误IP地址。国内用户访问 youtube 等网站时就会遇到这种情况

DNS 劫持是通过劫持特定的DNS服务器，在 DNS 服务器里存放错误的缓存信息，这样 DNS 服务器就会向用户返回错误的 IP 地址

对于 DNS 劫持，可用采用更换默认 DNS 服务器的方式解决。而 DNS 污染只能通过修改 hosts 文件、VPN 等方式绕过服务器

网关

在一个局域网内互相通信是不需要用到网关的，而局域网内的接口要访问外网时，则需要把请求发送到网关上，由网关负责向外发送流量。同理，外网的信息也是通过网关再传达给内网接口
